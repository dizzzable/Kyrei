# Kyrei v0.4.1 — устойчивые команды, Skills и восстановление работы

Этот выпуск делает длительную работу агента более прозрачной и надёжной: локальные навыки подключаются без ручных обходных путей, командные исследования получают проверяемые источники, а остановка и перезапуск не должны терять историю сессии.

## Главное

- **Skills из коробки.** Kyrei находит локальные `SKILL.md`, включая самодостаточные навыки без отдельной папки документации. Агент может сначала найти навык, а затем читать его частями, не переполняя контекст.
- **Навыки для команд.** Роли команд получают тот же управляемый доступ к Skills, поэтому исследователи, исполнители и тестировщики используют общий набор инструкций, а не зависят от случайного текста в чате.
- **Исследования с доказательствами.** Для веб-исследований сохраняются только источники, которые реально были успешно прочитаны агентскими инструментами: URL, заголовок, время и контрольный отпечаток содержимого. Модель не может подделать такой receipt в своём ответе.
- **Кэш исследований.** Команда повторно использует безопасные, нормализованные результаты текущей задачи и делает меньше одинаковых запросов.
- **Надёжная отмена.** На Windows остановка команды дополнительно завершает дерево дочерних процессов. Поздние сообщения после отмены или закрытия приложения не перезаписывают завершённую сессию.
- **Видимая работа.** Активность инструментов, задания и результаты команды синхронизируются с интерфейсом и сохраняются для восстановления после перезапуска.
- **Провайдеры и запуск.** Улучшены возврат к настройке провайдера, состояние защищённого хранилища и объяснения для первого подключения модели.

## Платформы

- Windows x64: Setup и Portable.
- macOS: DMG и ZIP для Intel и Apple Silicon.
- Linux x64/arm64: AppImage и DEB.
- Arch Linux x64: пакет `.pkg.tar.zst` для `pacman`.

Во вложениях есть `SHA256SUMS.txt` для проверки каждого установщика.

> Пакеты Windows пока не подписаны коммерческим сертификатом, а пакеты macOS не notarized. При первом запуске система может показать предупреждение: сверяйте SHA-256 из этого релиза.

## Проверка качества

- Полный gate: TypeScript, JavaScript, i18n и Vitest.
- Проверка зависимостей с порогом `npm audit --audit-level=high`.
- Сканирование коммитов на секреты перед нативной упаковкой.
- Кроссплатформенная упаковка на Windows, macOS и Ubuntu.

## Известные ограничения

- Веб-доступ остаётся инструментом агента для поиска и чтения публичных источников; пользовательского браузера в интерфейсе Kyrei нет.
- Полноэкранные терминальные TUI по-прежнему требуют отдельной PTY/ConPTY-интеграции.
