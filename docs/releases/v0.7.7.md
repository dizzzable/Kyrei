# Kyrei v0.7.7 — честные стримы и управление сессиями

Этот патч убирает ложное завершение длительных запросов и возвращает полное
управление сессиями прямо из боковой панели.

## Длительная работа агента

- Удалён внутренний Subscription Shield, который мог создавать собственный
  `AbortController`, таймер ожидания заголовков или таймер тишины стрима.
- Kyrei теперь передаёт поток провайдеру без промежуточного deadline: медленное
  reasoning или tool-вызов не считаются сбоем сами по себе.
- Старые сохранённые настройки Shield безопасно нейтрализуются; соответствующие
  контролы больше не отображаются в настройках.
- Отмена пользователем, настоящий сетевой сбой провайдера и failover остаются
  отдельными, корректно наблюдаемыми состояниями.

## Сессии

- В меню каждой сессии снова доступно **«Удалить навсегда»**.
- Перед удалением Kyrei показывает подтверждение с названием сессии.
- Архив остаётся обратимым: он скрывает чат из списка, сохраняя его для памяти.
- Постоянное удаление очищает локальный transcript и строки session-mirror FTS.

## Проверка

- Выполнены typecheck engine и renderer, проверки JavaScript и i18n, полный
  `npm run gate` и production-сборка `npm run package:prepare`.
- Добавлены регрессии для нативного stream-маршрута и передачи permanent-delete
  из activity rail в меню сессий.

## Обновление

Подтвердите обновление в Kyrei, когда приложение предложит новую версию. Релиз
содержит полные установщики для Windows, macOS, Linux и Arch Linux, а также
контрольные суммы SHA-256.
