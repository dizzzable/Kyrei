# Kyrei v0.7.3 — Reliable desktop release packaging

This corrective patch completes the v0.7.2 release path on every supported
desktop platform without changing user data or provider configuration.

## Packaging reliability

- Local native packaging reuses the Electron binary already installed by npm.
  This avoids a Windows filesystem lock during Electron extraction.
- Explicit architecture and cross-platform builds continue to use
  electron-builder's managed Electron download. This keeps GitHub Actions
  compatible with clean Windows, macOS, Linux x64 and Linux arm64 runners.
- The packaging contract is covered by tests so a local-only path cannot be
  accidentally configured globally again.

## Verification

- Full `npm run gate` passed: 237 test files, 1,701 tests passed, 4 skipped.
- Native Windows packaging completed successfully with `npm run dist:win`.
- The complete GitHub Actions package matrix passed for Windows x64, macOS
  x64/arm64, Linux x64 (AppImage/deb/pacman) and Linux arm64.

## Upgrade notes

Install through the in-app updater or download the installer for your
platform. Sessions, memories, provider secrets and workspace data remain
unchanged.
