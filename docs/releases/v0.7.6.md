# Kyrei v0.7.6 — устойчивый агент, память и инструменты

Этот патч устраняет несколько причин, из-за которых длительная работа агента
могла выглядеть как цикл, внезапно завершаться или терять полезный контекст.
Он также делает подключение MCP и настройку моделей предсказуемее.

## Длительные задачи и контекст

- Контекст передаётся между внутренними сессиями через структурированное
  сжатие и устойчивые записи памяти, а не обрывается при приближении к лимиту.
- Результат выполненного инструмента сохраняется как авторитетный факт для
  следующего шага. Это уменьшает повторные вызовы и ложные сообщения модели о
  том, что инструмент якобы не сработал.
- Улучшены маршрутизация дочерних агентов, обработка таймаутов и безопасное
  восстановление после временных сбоев провайдера.

## Память, MCP и провайдеры

- Встроенное файловое хранилище GBrain теперь надёжно создаётся при явной
  инициализации и имеет тестовое покрытие для fallback-сценария.
- Конфигурация MCP принимает длинные аргументы запуска: это важно для
  `node -e` и других локальных серверов с большим стартовым скриптом.
- Если пользователь точно указал MCP-сервер и инструмент, агент получает
  явную подсказку вызвать именно его. Вызовы, требующие подтверждения,
  по-прежнему не запускаются без разрешения пользователя.
- Для совместимых OpenAI-подобных endpoint'ов передаётся выбранный уровень
  reasoning/thinking. Интерфейс не притворяется, что видел рассуждение, если
  сам провайдер его не прислал в потоке.

## Интерфейс и наблюдаемость

- Уточнены подписи возможностей моделей, настройки reasoning и отображение
  ответов агента, чтобы технические события не засоряли обычный чат.
- Укреплены учёт использования, конфигурация провайдеров и диагностика
  состояния инструментов.

## Проверка

- `npm run gate`: 243 тестовых файла, 1 762 теста прошли; 4 пропущены.
- Выполнены typecheck engine и renderer, JavaScript- и i18n-проверки.
- `npm run package:prepare` успешно собрал renderer и engine.
- Отдельно проверены: передача `reasoningEffort`, обнаружение MCP-инструментов
  в запросе и автоматическое сжатие длинного контекста.

## Обновление

Автообновление скачает и установит подходящий пакет после подтверждения в
приложении. Установщики для Windows, macOS, Linux и Arch публикуются вместе с
SHA-256 контрольными суммами.
